# CartaVinAI Wireframes — Google UI

Wireframe React + Vite con UI minimale stile Google/Material.

## Caratteristiche

- React + Vite moderno
- CSS plain, nessun Tailwind
- Interfaccia bianca, pulita, con azione primaria blu
- Navigazione guidata a passaggi
- Logica aggiornata:
  - bottiglia consigliata per selezioni di più piatti / tavolo
  - singolo piatto solo per proposta al calice
- Report interno e versione cliente coerenti con questa logica
- Anteprima mobile QR

## Requisiti

Consigliato Node.js recente.

## Avvio

```bash
npm install
npm run dev
```

Apri l'URL mostrato da Vite, di solito:

```text
http://localhost:3000/
```

## Build

```bash
npm run build
npm run preview
```
